DEPLOYMENT INSTRUCTIONS
================================================================================

1. SET ENVIRONMENT VARIABLES
   - SUPABASE_URL
   - SUPABASE_SERVICE_ROLE_KEY
   - ANTHROPIC_API_KEY
   - EXERCISEDB_API_KEY (optional)

2. DEPLOY EDGE FUNCTIONS
   supabase functions deploy generate-program
   supabase functions deploy coach-send-message
   supabase functions deploy sync-exercises

3. SYNC EXERCISE DATABASE
   curl -X POST /functions/v1/sync-exercises \
     -H "Authorization: Bearer API_KEY"

4. INTEGRATE INTO APP
   - Import rulesEngine.ts and calculations.ts
   - Call edge functions from app
   - Display generated programs
   - Enable coaching interface

5. TEST ENDPOINTS
   - Test generate-program with test user
   - Test coach-send-message with sample messages
   - Test sync-exercises functionality
